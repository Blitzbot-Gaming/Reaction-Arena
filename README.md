# ⚡ Reaction Arena

A fast-paced reaction time game built using HTML, CSS, and JavaScript.

## 🎮 Features
- Real-time reaction tracking
- Combo multiplier system
- Dynamic difficulty (speed scaling)
- Particle effects on hit
- Sound feedback system
- Persistent leaderboard (localStorage)

## 🧠 Tech Stack
- HTML
- CSS
- JavaScript (DOM + Game Logic)

## 🚀 How to Play
1. Click "Start Game"
2. Click the target as fast as possible
3. Build combos for higher scores
4. Try to beat your high score

## 📌 Future Improvements
- Online leaderboard (Firebase)
- Mobile version (React Native)
- Power-ups system
